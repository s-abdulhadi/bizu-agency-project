# Bizu Agency - Design Specification

## Colors
### Primary Palette
- **Primary Purple**: `#6C5CE7` (Vibrant, creative, energetic)
- **Primary Accent**: `#A29BFE` (Softer purple for hover/backgrounds)
- **Secondary Teal**: `#00CEC9` (Fresh, modern contrast)

### Neutrals
- **Background**: `#F9FAFB` (Very light gray, almost white)
- **Surface**: `#FFFFFF` (Pure white for cards/sections)
- **Text Dark**: `#2D3436` (Soft black for readability)
- **Text Light**: `#636E72` (Gray for subheadings/microcopy)
- **Border**: `#DFE6E9` (Subtle dividers)

### Status
- **Success**: `#00B894`
- **Error**: `#FF7675`
- **Warning**: `#FDCB6E`

## Typography
**Font Family**: 'Outfit' (Google Font) - Modern, geometric, accessible.
Fallback: 'sans-serif'.

### Hierarchy
- **Hero/Display**: 700 Bold, 3.5rem (56px)
- **H1**: 700 Bold, 2.5rem (40px)
- **H2**: 600 SemiBold, 2rem (32px)
- **H3**: 600 SemiBold, 1.5rem (24px)
- **Body**: 400 Regular, 1rem (16px), 1.6 line-height
- **Small/Micro**: 400 Regular, 0.875rem (14px)

## Layout & Grid
- **Container**: Max-width `1200px`, centered.
- **Padding**: `1.5rem` (24px) on mobile, `2rem` (32px) on desktop.
- **Grid Gap**: `2rem` (32px).

### Breakpoints
- **Mobile**: < 768px
- **Tablet**: 768px - 1199px
- **Desktop**: >= 1200px

## Shadows & Radius
- **Card Shadow**: `0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06)`
- **Hover Shadow**: `0 10px 15px -3px rgba(0, 0, 0, 0.1), 0 4px 6px -2px rgba(0, 0, 0, 0.05)`
- **Border Radius**: 
  - Buttons/Cards: `8px` or `12px`
  - Inputs: `6px`

## UI Elements
- **Buttons**:
  - Primary: Solid Purple, White Text, Hover lift.
  - Ghost/Outline: Transparent background, Purple Border.
- **Inputs**: Light gray border, focus ring with Primary Purple shadow.
